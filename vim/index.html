Vim tips and tricks

Vim is the best text editor out there, if you know how to use it! It has a steep learning curve, but if you can master it, then you will be very fast at editing text.

The basics

How do I quit vim?

  • To quit vim, type 
    :q
    (q for quit) and press the return key
  • To save and quit, type 
    :wq
    (w for write, q for quit) instead
  • To quit without saving, type 
    :q!

Entering insert mode

I use these keys all the time! 
  • i
    enters insert mode at the cursor
  • I
    enters insert mode at the beginning of the line
  • a
    enters insert mode at the character after (a for after) the cursor
  • A
    enters insert mode at the end of the line (A for after the line)
  • o
    adds a new line under (o for open line) the current line and enters insert mode
  • O
    adds a new line above the current line and enters insert mode

Text objects

A feature of vim that is very powerful is that it allows you to be quite specific about what you select with very few keypresses. These all work with 

d, x, y, c, v
and others - the examples will use y (for yank (copy))

  • yaw
    - yank a word - yanks a word and the space after it
  • yiw
    - yank in word - yanks a word without the space
  • ye
    - yanks to the end of the word
  • yb
    - yanks from the beginning of the current word
